# DSPMS
Dead Simple Parcel Management System

### Description
a quick basic api for basic operation of PMS

backend: using Horus.php framework mainly for routing purpose

frontend: tba

### API LIST

##### user login
> POST | /api/login

##### user read all
> GET | /api/user

##### user create
> POST | /api/user

##### user update all
> POST | /api/user/{id}

##### user DELETE all
> DELETE | /api/user/{id}
